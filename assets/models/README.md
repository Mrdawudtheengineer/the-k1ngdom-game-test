# Asset Sources

Use these free CC0 assets and convert to `.glb` when possible. Place models into the folders below.

## Environment & Buildings
- https://polyscann.com/assets/14 (medieval stone house with PBR textures)
- https://3dmodelscc0.itch.io/free-cc0-medieval-props (barrels, crates, scrolls, props)

## Characters / NPCs
- https://free-game-assets.itch.io/free-medieval-3d-people-low-poly-pack
- https://sketchfab.com/3d-models/fantasy-villager-10-1154cacb04ef4057ba379138532bcb51

## Folder Layout
```
/assets/models/characters/
  villager01.glb
  merchant.glb
  guard.glb
```
